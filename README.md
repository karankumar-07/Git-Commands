# Git-Commands

## Initialization

`git int`

## Git Add and Commit

`git add .`
`git commit -m ""`

## Git Push

`git push`

## Git show logs

`git log`

## Git Change to Previous Commit

`git checkout <commit-id>`

## Git List Branch

`git branch`

## Git Create New Branch

`git branch <name>`

## Git Create New Branch and also Switch to New Branch

`git checkout -b <name>`
